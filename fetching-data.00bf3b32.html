Fetching Data

Once authenticated, you can fetch user data using a few simple arguments.

const results = await hashcache.fetch({ tags, limit, olderThan, newerThan }, (docs) => {
    // ... process decrypted documents ...
})

Let's break this out into a bit more detail and with examples.

For instance, to find the last 20 items tagged with "notes", you would do the following.

hashcache.fetch({
    tags: [
        'tag', // array of strings
    ],
    olderThan: Date.now(), // default, data is fetched in desc order of 'created' timestamp
    newerThan: 0, // default
    limit: 20, // default (max)
})
