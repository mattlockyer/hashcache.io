Getting Started

Hashcache can be imported using a bundler or a script tag.

Using a bundler:

npm i hashcache
...
import { hashcache } from 'hashcache'

Using a script tag:

<script src="https://hashcache.io/sdk"></script>

After you should have the hashcache object available in your module or window scope. Try it out.

console.log(hashcache.version)
