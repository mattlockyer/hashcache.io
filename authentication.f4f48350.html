Authentication

Users must authenticate with your app before they can fetch and store data.

You can manually register a user or simply call login and have them set their Data Two Factor Authentication (D2FA) password with Hashcache.

If this is the first time login (registration) you will want to enforce a backend check.

Thankfully you can easily do this on your backend by requesting a magic link from the Hashcache API that you can send in an email to the registering user. The link can redirect the user to your app and complete the registration flow.

hashcache.login(username)

The user will be presented with a Hashcache D2FA dialog. If this user has never registered before they will an additional field to confirm their D2FA.

Once authenticated, the user can now fetch and store data.
